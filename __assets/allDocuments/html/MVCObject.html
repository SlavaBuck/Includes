Class: MVCObject

MVCObject

new MVCObject(id)

Базовый MVC-Объект.

Родительский класс для всех MVC-объектов (MVCModel, MVCView, MVCController, MVCApplication)

ПРИМЕЧАНИЕ: Поскольку MVCObject.prototype является родительским для всех основных MVC-объектов, его расширение позволит также расширять и их.

Parameters:
Name Type Argument Description
id object | string <optional>

Если параметр представлен текстовой строкой - она присваивается собственному свойству id, в случае если в аргументе представлен объект - экземпляр MVCObject расширется свойствами этого объекта (используется метод слияния extend())
Properties:
Name Type Description
id string

id (идентификатор) объекта, если не указан в аргументах конструтора - инициализируется пустой строкой "";
Example
var obj = new MVCObject ("obj"); // obj -> { id:"obj" }
var obj = new MVCObject ({ text:"obj1"}); // obj -> { id:"", text:"obj1" }
var obj = new MVCObject ({ id:"obj", text:"obj1"}); // obj -> { id:"obj", text:"obj1" }
