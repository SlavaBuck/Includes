Module: MVC

MVC

Базовый модуль библиотеки MVC. Представляет основные классы для реализации приложения согласно архитектурному паттерну MVC.

Properties:
Name Type Description
MVC.app MVCApplication

Указатель на экземпляр MVCApplication (переустанавливается в конструкторе MVCApplication() и, таким образом, в рамках скрипта указывает на последний инициированный(активный) объект-приложение.
MVC.fastmode MVCApplication

Специальный флаг (по умолчанию равен false), если true - проверки на уникальность id при добавлении MVC-объектов в коллекции MVCApplication.models, MVCApplication.views и MVCApplication.controllers не производится. Может быть полезна для интенсивной работы с большими коллекциями при условии реализованного в программе надёжного алгоритма для формирования уникальных id-шников.
