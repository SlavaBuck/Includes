makePng(dim, rgb) → {ScriptUIImage}

Формирует изображение в виде прямоугольника с заданным цветом и возвращает его в формате изображения ScriptUIImage, пригодном для отображения в ScriptUI - элементах управления.

ПРИМЕЧАНИЕ: Основано на A Jongware Product. Оргинальная функция строит отображаемый в память битовый графический образ изображения в формате PNG на основе "cырой" ресурсной байт-строки.

Parameters:
Name Type Description
dim number | Array.<number>

Размер изображения: Число (формируется квадрат) или массив из двух целых [width, hight] (ширина и высота прямоугольника).
rgb Array.<number>

Цвет в формате RGB - массив из тёрх целых [R, G, B] от 0 до 255
Returns:

Образ PNG-изображения в формате ScriptUIImage - пригодном для использования в ScriptUI

Type
ScriptUIImage
Example
var cRed = [255, 0, 0];
var img = makePng(64, cRed);         // -> 64x64 image
var img = makePng([64, 32], cRed);   // -> 64x32 image
