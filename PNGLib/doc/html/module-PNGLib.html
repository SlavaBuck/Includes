PNGLib

Формирование ScriptUIImage изображений в памяти для использования с графическими элементами управления ScriptUI

Добавляет в глобальное пространство имён метод makePng(). В собственном пространство имён расширяет группой вспомогательных методов, например adler32(), crc32s(), i2s() и др...
