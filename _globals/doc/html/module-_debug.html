Module: _debug

_debug

Расширяет глобальное пространство имён набором функций:

  • log() вывод в консоль;
  • _debug() синоним inspect(), вывод подробной информации об объекте;
  • trace() вывод подробной информации об ошибке, включая стек и т.д.;
  • assert() простая функция тестирования;
  • _timer() простой таймер для замера времени выполнения кода.
