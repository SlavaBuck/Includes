Class: _timer

_timer

new _timer()

Простейший мини-таймер для расчёта времени выполнения операций:

Methods

start()

Фиксирует текущее (стартовое) время и возвращает его в миллисекундах

stop()

Фиксирует текущее (конечное) время и возвращает разницу между старотовым временем в миллисекундах

toString()

Простое форматирование вывода - возвращает строку ххх ms., где ххх - значение миллисекунд внутреннего счётчика
