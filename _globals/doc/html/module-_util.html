Module: _util

_util

Минимальный набор вспомогательных функций для работы с объектами и коллекциями javascript, без которых не обходится ни один модуль в сборке.

Подключается модуль _debug и, дополнительно, в глобальное пространство имён добавляет методы для работы с:

Объектами:

  • extend() расширение объекта, слиянием свойств;
  • merge() получение нового объекта, путём объединения свойств;
  • clone() полное клонирование объекта;
  • defaults() расширение объекта только отсутствующими в нём свойствами;
  • inherit() реализация прототипоного наследования;
  • keys() получение списка свойств объекта в виде массива (Array)

    Архивами и Коллекциями:

  • each() синоним forEach(), итерация;
  • indexOf() получения индекса элемента в массиве;

    Функциями:

  • bind() привязка функции к контексту;

    Проверка типа:

  • classof() возвращает текстовую строку с наименованием класса/типа объекта;
  • isArray() проверяет, является ли объект массивом;
  • isLikeArray() проверяет, является ли объект, подобным массиву (например, коллекцией).

    Дополнительно:

  • doBatFile() выполнение команд DOS/Windows;
  • openURL() открытие URL в браузере.
