# UIColors
***Status:*** *Редактируемая версия...* | ***Version:*** *0.1* | ***Last update:*** *25.05.2014*

Работа с цветом в формате RGB и RGBA

Содержит базовый набор цветовых констант COLORS (в uint формате) и набор функций для работы с ними:

* toRGB();
* toRGBA();
* RGBtoRGBA();
* RGBAtoRGB();
* RGBtoValue();
* RGBAtoValue();
* parseColor();
