Module: DOM

DOM

Дополняет объект MVCDocument к базовому набору MVC-классов и расширяет базовый класс MVCApplication для включения в приложение поддержку фрхитектуры многодокументных интерфейсов:

  • MVCApplication.registerDocumentFactory(docFactory, docsView)
  • MVCApplication.addDocument(obj)
  • MVCApplication.getDocumentByID(id)
  • MVCApplication.getDocumentByName(name)
  • MVCApplication.closeDocument(doc)
  • MVCApplication.loadDocument()
  • MVCApplication.saveDocument()
  • MVCApplication.saveAllDocument()
  • MVCApplication.saveAsDocument()
